# VR识别插件

## 一、功能介绍：

VR识别插件，EasyVR，可用于网页，微信小程序，手机app外置等；

## 二、使用方法：

通过摄像头扫描指定图案完成图片识别，即可将动态3d模型展示在画面中，与现实场景融合，为用户提供VR体验。
